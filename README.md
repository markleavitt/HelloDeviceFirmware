Binary files for OTA update of Hello Everyday device firmware

HelloCell.ino.bin supports Hello-Everyday and Hello-Everyday Plus devices

HelloTunes1.ino.bin supports the beta/pilot Hello Tunes devices (with internal MP3 player)

HelloTunes2.ino.bin will support the production Hello Tunes devices (with internal SD card)

HelloDevice.ino.bin and HelloDevice.json are OBSOLETE (supported early WiFi device)

Bootloader files are NOT obsolete:  HelloDevice.ino.bootloader.bin, HelloDevice.ino.partitions.bin, boot_app0.bin, tinyuf2.bin are STILL NEEDED
